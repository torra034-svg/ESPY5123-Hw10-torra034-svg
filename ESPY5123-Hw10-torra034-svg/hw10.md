# Part 3: Build a Logistic Regression Classifier

## 3.1 Specify the model

Create a **logistic regression specification** using `tidymodels`.

``` r
log_spec <- logistic_reg() %>% 
  set_engine("glm") %>% 
  set_mode("classification")
```

Create a workflow that:
